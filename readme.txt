Copy one of the following shorcodes into any post inside Wordpress visual editor:

*[comp_company_login]*

*[nayak_searchboxes options='{"size":"160x600"}']*

*[comp_create_company]*

*[comp_company_listing]*

To add a demo output, footer or sidebar, open the template file in Wordpress editor and copy the following code in it (based on the sice you want to use):

*`<?php echo do_shortcode('[comp_company_login]'); ?>`*

*`<?php echo do_shortcode('[comp_company_listing]'); ?>`*

*`<?php echo do_shortcode('[comp_create_company]'); ?>`*
